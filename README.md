# Mecanografia

Aplicación web estática para visualizar el progreso de prácticas de mecanografía.

## Funcionalidades

- Historial de sesiones con fecha, nivel, modo, duración, PPM, WPM, precisión y errores.
- Mejores marcas de PPM, WPM y racha de caracteres correctos.
- Evolución visual de PPM y WPM en un gráfico de líneas.
- Precisión media, errores frecuentes y tiempo total de práctica.
- Filtros combinables por nivel, modo, rango de fecha y duración.

## Desarrollo

```bash
npm run start
```

## Comprobaciones

npm run check
